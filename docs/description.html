This plugin allows administrators to leave their messages for users on Front End block

There are several styles, available for Announcements block:

  • an--info
  • an--success
  • an--important
  • an--danger
  • an--inverse

To change style, go to Admin Dashboard -> Content -> Blocks. Click on Edit Announcements block. In the 'CSS class name' config field, enter desired classname like this: "an an--success" (without quotes).
